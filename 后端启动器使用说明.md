# SystemMonitor 后端启动器使用说明

## 功能介绍
SystemMonitor 后端启动器是一个图形化工具，用于方便地启动和管理 SystemMonitor 后端服务器。

## 使用方法

### 1. 启动软件
双击 `BackendLauncher.exe` 启动后端启动器。

### 2. 配置服务器
在"服务器配置"区域，您可以设置：
- **主机地址**: 默认为 `0.0.0.0`，表示监听所有网络接口
- **端口号**: 默认为 `8000`
- **启动后自动打开浏览器**: 勾选此项后，服务器启动成功会自动打开管理界面

### 3. 启动/停止服务
- 点击"启动服务"按钮启动后端服务器
- 点击"停止服务"按钮停止后端服务器
- 服务状态会实时显示在"服务状态"区域

### 4. 访问管理界面
- 本地访问地址: `http://127.0.0.1:8000`
- 局域网访问地址: `http://[您的IP地址]:8000`
- 点击"打开管理界面"按钮可直接打开浏览器访问管理界面

### 5. 保存配置
点击"保存配置"按钮可将当前配置保存到 `backend_config.json` 文件中，下次启动时会自动加载。

## 注意事项
1. 确保端口未被其他程序占用
2. 如果修改了端口号，客户端也需要相应修改连接地址
3. 启动器会自动检测服务器状态，即使不是通过启动器启动的服务器也能检测到
4. 关闭启动器时，如果服务器正在运行，会提示是否停止服务器

## 文件结构
```
system-monitor/
├── dist/
│   ├── BackendLauncher.exe  # 后端启动器
│   └── SystemMonitor.exe    # 客户端程序
├── backend/                 # 后端代码目录
├── backend_config.json      # 后端配置文件（自动生成）
└── backend_launcher.py      # 后端启动器源代码
```

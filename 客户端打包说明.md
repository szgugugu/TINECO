# SystemMonitor 客户端打包说明

## 打包结果

客户端已成功打包为单个可执行文件：`dist/SystemMonitor.exe`

## 使用方法

### 方法一：直接运行exe文件
1. 双击 `dist/SystemMonitor.exe` 运行客户端
2. 客户端将尝试连接到默认服务器地址（http://localhost:8000）

### 方法二：通过命令行指定服务器地址
```
dist\SystemMonitor.exe --server http://192.168.1.100:8000
```

### 方法三：修改配置文件
1. 修改 `client_config.py` 文件中的 `SERVER_URL` 变量
2. 然后运行 `dist\SystemMonitor.exe`

## 注意事项

1. **确保后端服务器已启动**：在运行客户端之前，请确保后端服务器（运行 start_backend.py）已启动并可访问。

2. **网络连接**：如果客户端和服务器不在同一台计算机上，请确保：
   - 防火墙允许8000端口的通信
   - 服务器地址正确（使用服务器的实际IP地址）

3. **依赖项**：exe文件已包含所有必要的依赖项，无需额外安装Python或相关库。

4. **配置文件**：`client_config.py` 文件需要与exe文件在同一目录下，或者可以通过命令行参数指定服务器地址。

## 文件说明

- `dist/SystemMonitor.exe`: 主程序文件
- `client_config.py`: 配置文件（可选）
- `run_client.bat`: 启动脚本（可选）
- `build_exe_simple.bat`: 打包脚本

## 故障排除

如果遇到问题，请检查：

1. 服务器是否正常运行
2. 网络连接是否正常
3. 服务器地址是否正确
4. 防火墙设置是否阻止了连接

# 系统信息监测工具 v1.6.0

一个简洁美观的Windows系统监控工具，提供系统信息显示、性能监控和网络连通性测试功能。采用现代化UI设计，界面紧凑高效。

## 功能特点

### 界面设计
- **三列布局**：系统信息、性能监控、网络测试整合在同一界面，无需切换标签页
- **现代化UI**：采用企业级配色方案，深色主题，专业美观
- **紧凑布局**：优化字段间距，信息密度高，界面整洁
- **响应式设计**：自适应不同DPI和窗口大小
- **高DPI支持**：完美支持高分辨率显示器

### 系统信息显示
- 当前用户名
- 计算机名称  
- 操作系统版本
- 网络适配器信息（IP地址和MAC地址）
- 实时系统运行时间

### 性能监控
- **CPU占用率**：实时显示CPU使用百分比，带进度条可视化
- **内存占用率**：实时显示内存使用情况，进度条直观展示
- **自动刷新**：每2秒自动更新性能数据
- **多线程处理**：不影响系统性能的后台监控

### 网络连通性测试
- **多目标测试**：支持百度、TikTok、谷歌DNS、8.8.8.8等多个测试目标
- **详细测试结果**：显示连通状态、响应时间、丢包率
- **快速测试**：每次5次ping，快速获取网络状况
- **进度指示**：实时显示测试进度
- **并发测试**：多线程并发测试，提高效率
- **平均值统计**：测试结束后显示平均延迟和平均丢包率
- **一键停止**：支持随时停止所有测试，界面保持响应

### 截图功能
- **一键截图**：点击截图按钮即可截取整个软件窗口
- **剪贴板集成**：截图后自动复制到剪贴板，支持Ctrl+V粘贴
- **高清截图**：保持原始分辨率，图片清晰度高

## 系统要求

- Windows 10/11 (推荐)
- Python 3.7+ (开发环境)
- 运行环境：可直接使用打包好的exe文件

## 安装与使用

### 方法1：直接运行exe文件（推荐）

1. 找到 `dist` 文件夹中的 `system_monitor.exe`
2. 双击运行即可使用，无需安装任何依赖

### 方法2：源码运行

1. 确保已安装Python 3.7+
2. 安装依赖库：
   ```bash
   py -m pip install psutil pillow
   ```
3. 运行程序：
   ```bash
   py system_monitor.py
   ```

### 方法3：使用批处理文件

双击运行"启动系统监控.bat"文件，自动处理环境和依赖。

## 界面说明

程序界面分为三个主要区域：

### 左侧 - 系统信息卡片
- 用户信息和系统详情
- 网络适配器配置
- 系统运行状态

### 中间 - 性能监控卡片  
- CPU使用率实时监控
- 内存使用率实时监控
- 进度条可视化显示

### 右侧 - 网络测试卡片
- 网络连通性测试控制
- 多目标测试结果展示
- 响应时间和丢包率统计

## 操作说明

### 启动程序
- 双击exe文件或运行Python脚本即可启动
- 程序会自动适应屏幕DPI设置

### 查看系统信息
- 启动后自动显示系统基本信息
- 网络信息会实时更新

### 性能监控
- CPU和内存数据每2秒自动刷新
- 通过进度条直观显示使用情况

### 网络测试
1. 点击"开始测试"按钮
2. 等待所有目标测试完成或点击"停止测试"按钮
3. 查看详细的测试结果：
   - **连通状态**：✅连通 / ❌不通 / ⚠️不稳定
   - **响应时间**：平均ping延迟（毫秒）
   - **丢包率**：测试期间的数据包丢失百分比
   - **平均延迟**：从开始测试到结束测试之间的平均延迟（测试结束后显示）
   - **平均丢包**：从开始测试到结束测试之间的平均丢包率（测试结束后显示）

### 截图功能
1. 点击"截图"按钮
2. 程序会自动截取整个窗口并复制到剪贴板
3. 在其他应用程序中按Ctrl+V即可粘贴截图

## 常见问题

**Q: 程序无法启动怎么办？**
A: 确保系统为Windows 10/11，如使用源码运行请检查Python环境和依赖库是否正确安装。

**Q: 网络测试结果显示异常？**
A: 某些网络环境可能需要管理员权限才能获取完整的ping统计信息。

**Q: 截图功能无法使用？**
A: 请确保已安装Pillow和pywin32依赖库，如使用exe文件则无需担心。

**Q: 点击停止按钮后界面卡顿？**
A: v1.6.0版本已优化停止功能，采用异步方式处理，界面不会卡顿。

**Q: 界面显示不完整？**  
A: 程序已支持高DPI显示，如仍有问题请尝试调整系统缩放设置。

**Q: 如何获取技术支持？**
A: 本工具为开源项目，可在项目仓库提交问题反馈。

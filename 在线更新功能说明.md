# 系统监控工具 - 在线更新功能使用说明

## 新增功能：在线更新

系统监控工具现已支持在线更新功能，方便用户获取最新版本。

### 使用方法

1. **启动应用程序**
   - 双击运行 `start_monitor.bat` 文件启动系统监控工具

2. **检查更新**
   - 在应用程序主界面的右上角，点击"检查更新"按钮
   - 系统将自动连接到更新服务器检查是否有新版本

3. **更新过程**
   - 如果有新版本，将显示更新对话框，包含新版本信息和更新说明
   - 点击"立即更新"开始下载新版本
   - 下载过程中将显示进度条和下载状态
   - 下载完成后，系统将自动安装更新并重启应用程序

4. **稍后更新**
   - 如果不想立即更新，可以点击"稍后更新"按钮
   - 更新提醒将在下次启动时再次显示

### 注意事项

- 更新过程需要网络连接
- 更新过程中请勿关闭应用程序
- 更新完成后应用程序将自动重启
- 如果更新失败，请检查网络连接或联系技术支持

### 技术细节

- 更新文件通过安全连接下载
- 更新过程会自动备份当前版本
- 如果更新失败，系统将恢复到原版本
